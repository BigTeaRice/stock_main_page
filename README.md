# 股票分析自动化系统

## 功能特性
1. **多数据源支持**：
   - 国际股票（Yahoo Finance）
   - A股（AkShare）
   - 模拟数据（测试模式）

2. **自动化分析**：
   - 实时数据抓取
   - 技术指标计算（MA/RSA/MACD/Bollinger Bands）
   - 涨跌幅预测

3. **可视化展示**：
   - 交互式K线图
   - 动态技术指标面板
   - 多维度数据对比

4. **自动化部署**：
   - GitHub Actions 定时触发
   - 结果自动上传至GitHub Pages
